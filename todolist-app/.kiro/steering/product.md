# Product: Todolist App

A simple browser-based todo list application. Users can add, edit, delete, and filter tasks. All data persists locally via `localStorage` — no backend, no login.

## Core Features
- Add tasks with title validation (no empty/whitespace-only titles)
- Mark tasks complete/incomplete (toggle)
- Inline edit task titles
- Delete individual tasks or bulk-clear all completed tasks
- Filter tasks by status: All / Active / Completed
- Auto-save to `localStorage` on every change; auto-load on page refresh

## Target Users
Individual users managing personal daily tasks in a web browser.

## Language
The project documentation and UI are written in **Bahasa Indonesia**.
